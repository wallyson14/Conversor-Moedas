# Conversor de Moedas 💰

Um conversor de moedas simples e responsivo que converte Dólar, Euro e Libra em Reais.

## 🚀 Funcionalidades
- Conversão em tempo real
- Interface moderna e limpa
- Moedas suportadas: USD, EUR, GBP
- Design responsivo

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript
- Google Fonts
